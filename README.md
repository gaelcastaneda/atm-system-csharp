# ATM System in C# (Windows Forms)

Graphical ATM simulation developed in C# using Windows Forms and object-oriented programming. The application allows users to perform common banking operations such as checking balance, depositing money, withdrawing funds, and navigating through a user-friendly interface.

## Features

* Graphical user interface built with Windows Forms.
* Balance inquiry.
* Cash deposit.
* Cash withdrawal.
* Input validation.
* Navigation between multiple forms.
* Banking transaction simulation.

## Technologies Used

* C#
* .NET Framework
* Windows Forms
* Object-Oriented Programming (OOP)

## Project Structure

* `Program.cs` – Application entry point.
* `Menu.cs` – Main ATM menu.
* `Custodio.cs` – Data and account management.
* `Ingcus.cs` – Deposit functionality.
* `Ingretiro.cs` – Withdrawal functionality.
* `Form1.cs` – Initial interface.
* `App.config` – Application configuration.
